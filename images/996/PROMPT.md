# 996 科研包身工 — 文生图 Prompt

## 统一风格基线

**基础正向 Prompt：**
Q版导师人格角色卡，夸张可爱的人物设计，轻3D软塑料/盲盒质感，表情生动戏剧化，服装和道具强烈服务角色设定，居中构图，1:1方图，干净简洁背景带轻渐变，色彩鲜明饱和但不过曝，幽默搞笑风格，略带学术圈黑色幽默气质，高细节高完成度，无文字无水印

**负向 Prompt：**
不要真实摄影风格，不要多人物，不要复杂背景场景，不要赛博霓虹堆砌，不要任何文字/水印/logo，不要模糊低清，不要恐怖暗黑风，不要畸形肢体，不要写实人脸

---

## 角色 Prompt

```
Q版导师人格角色卡，"科研包身工"——把学生当劳动力的导师，像包工头站在堆积如山的项目文件和横向合同上，一手拿任务分配表一手指向远方的deadline，身后是小小学生剪影在搬运实验器材和文档，周围漂浮着"横向项目""不准实习""周末加班"标签，表情理所当然，背景工业灰和橙色渐变带工地感，Q版夸张盲盒质感，轻3D软塑料材质，讽刺搞笑，居中构图，1:1，无文字无水印
```

## 关键视觉元素

- 包工头气质（安全帽 or 指挥姿态）
- 堆积如山的项目文件/合同
- 身后小小学生剪影搬东西
- 任务分配表 + deadline 指向
- 背景：工业灰→橙色渐变，工地/流水线感
