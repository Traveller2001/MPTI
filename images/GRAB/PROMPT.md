# GRAB 摘桃圣手 — 文生图 Prompt

## 统一风格基线

**基础正向 Prompt：**
Q版导师人格角色卡，夸张可爱的人物设计，轻3D软塑料/盲盒质感，表情生动戏剧化，服装和道具强烈服务角色设定，居中构图，1:1方图，干净简洁背景带轻渐变，色彩鲜明饱和但不过曝，幽默搞笑风格，略带学术圈黑色幽默气质，高细节高完成度，无文字无水印

**负向 Prompt：**
不要真实摄影风格，不要多人物，不要复杂背景场景，不要赛博霓虹堆砌，不要任何文字/水印/logo，不要模糊低清，不要恐怖暗黑风，不要畸形肢体，不要写实人脸

---

## 角色 Prompt

```
Q版导师人格角色卡，"摘桃圣手"——抢夺学生成果的导师，一手高举闪光论文奖杯，另一手从身后果实累累的科研树上摘果子，树下有小学生辛勤浇水施肥的痕迹，身边漂浮着被修改署名的论文页面和"我们团队的成果"气泡，表情得意洋洋，背景金色暗绿渐变，Q版夸张盲盒质感，轻3D软塑料材质，讽刺搞笑，居中构图，1:1，无文字无水印
```

## 关键视觉元素

- 一手举奖杯/论文，一手从树上摘果子
- 科研果树（果实 = 论文/成果）
- 树下学生浇水施肥的痕迹（小铲子、水壶）
- 被修改署名的论文页面漂浮
- 表情：得意洋洋、志得意满
- 背景：金色→暗绿渐变
