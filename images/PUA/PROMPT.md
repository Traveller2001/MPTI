# PUA 宗师 — 文生图 Prompt

## 统一风格基线

**基础正向 Prompt：**
Q版导师人格角色卡，夸张可爱的人物设计，轻3D软塑料/盲盒质感，表情生动戏剧化，服装和道具强烈服务角色设定，居中构图，1:1方图，干净简洁背景带轻渐变，色彩鲜明饱和但不过曝，幽默搞笑风格，略带学术圈黑色幽默气质，高细节高完成度，无文字无水印

**负向 Prompt：**
不要真实摄影风格，不要多人物，不要复杂背景场景，不要赛博霓虹堆砌，不要任何文字/水印/logo，不要模糊低清，不要恐怖暗黑风，不要畸形肢体，不要写实人脸

---

## 角色 Prompt

```
Q版导师人格角色卡，"PUA宗师"——精神控制型导师，表面温文尔雅但眼神带操控感，一只手伸出来像要拍肩鼓励，另一只手背后握着提线木偶控制杆，身边漂浮着矛盾的对话气泡（"你很有潜力"和"你看看别人"同时存在），周围有隐约的牵引线和阴影，表情复杂像笑又像审视，背景浅紫暗红渐变带压迫感但不恐怖，Q版夸张盲盒质感，轻3D软塑料材质，搞笑讽刺，居中构图，1:1，无文字无水印
```

## 关键视觉元素

- 提线木偶控制杆（背在身后）
- 矛盾对话气泡（一句夸一句贬）
- 隐约的牵引线/丝线
- 表情：表面微笑但带审视感
- 背景：浅紫→暗红渐变，压迫感
