# 图片放置说明

当前 `MPTI` 页面已经支持无图占位，但正式上线前建议按以下规则放入插画：

- 目录规则：`images/<CODE>/<CODE>.webp`
- 例如：`images/KING/KING.webp`
- 隐藏人格与兜底人格同样遵守该规则：
  - `images/GHOST/GHOST.webp`
  - `images/404/404.webp`

建议统一出图规格：

- 画幅：`1:1`
- 背景：简洁渐变或单色背景
- 构图：单角色居中
- 格式：优先 `webp`
- 风格：参考 [docs/IMAGE_PROMPTS.md](../docs/IMAGE_PROMPTS.md)
