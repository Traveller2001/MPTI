# MPTI 图片 Prompt

## 统一风格基线

建议所有人格图统一使用同一套基础风格，保证图鉴和结果页看起来是一个系列：

**基础正向 Prompt：**

`单角色导师人格海报，Q版夸张人物设计，轻微 3D 软塑料质感，表情戏剧化，服装和道具强烈服务人物设定，居中构图，1:1 方图，背景简洁干净带轻渐变，色彩鲜明但不过曝，像一套统一的人格测试角色卡，幽默、机灵、略带学术圈黑色幽默，高细节，高完成度，无文字，无水印`

**负向 Prompt：**

`不要真实摄影，不要多人物，不要复杂场景，不要赛博霓虹堆砌，不要文字，不要水印，不要模糊，不要恐怖，不要畸形肢体，不要低清晰度`
